#search
`/[SEARCH_STRING]` and press `Enter` to start searching. Press `n` to find next match.

#delete-all
`:%d`
* `:` enters command mode
* `%` select all lines
* `d` delete

#common-operations
* `u` undo
* `yy` yank(copy)
* `p` paste
* `dd` delete one line
* `cw` change word
